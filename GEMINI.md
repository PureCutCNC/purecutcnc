# Gemini CLI Mandate

This project uses `ARCHITECTURE.md` as its primary source of truth for architectural vision, coding standards, and operational constraints. 

## Primary Directives:
1. **Source of Truth:** Always refer to `ARCHITECTURE.md` for foundational project rules.
2. **Feature Context:** Always check the `planning/` directory for detailed implementation plans and design documents before starting work on a feature.
3. **Precedence:** Instructions in `ARCHITECTURE.md` and `planning/*.md` take absolute precedence over general defaults.
